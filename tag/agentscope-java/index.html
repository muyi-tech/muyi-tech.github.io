AutoContextMemory 压缩机制管理

AgentScope Java 上下文自动压缩详解

🎯 核心目的

解决上下文溢出问题：当对话历史超过模型 token 限制时，自动压缩历史消息，保留关键信息，避免上下文无限增长。

🔧 核心配置参数

AutoContextConfig config = AutoContextConfig.builder()
    .tokenRatio(0.4)      // 压缩比例：保留 40% 的 token
    .lastKeep(10)         // 保留最近 10 条消息不压缩
    .build();
